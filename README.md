## Cara Run
#### 1. Clone Repository
`git clone https://github.com/luthfinovra/Tubes_Jarkom.git`

#### 2. Pindah ke Folder Tubes
`cd Tubes_Jarkom/Tubes\ Jarkom/`

#### 3. Run Server
* Windows
`python server.py`
* MacOS / Linux
`sudo python server.py`

#### 4.1. Buka WebServer
`http://localhost:80`

#### 4.2. Buka client.py
##### Buka terminal dan masuk ke directory dari repository yang sudah di clone, kemudian run :
`sudo python client.py`

## Fitur Yang ada Pada Server
#### Server dapat diakses menggunakan browser atau client.py. Disarankan untuk menggunakan incognito dalam browser dan run as administrator / sudo untuk menjalankan client.py
### Run Menggunakan Browser
#### Pertama run server dengan cara yang sudah dideskripsikan diatas. Kemudian buka browser (disarankan incognito) kemudian buka link
`http://localhost:80`
